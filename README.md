# Telegram Investing Bot (텔레그램 주식 종목 검색 봇)

이 프로젝트는 텔레그램 메신저 채팅방에서 `/get <종목명>` 명령어를 통해 Investing.com의 종목 실시간 검색 결과(종목명, 티커, 홈페이지 링크)를 바로 받아볼 수 있게 해주는 파이썬 기반 텔레그램 봇입니다. 

---

## 🛠 처음 설치 방법 가이드 (Installation Guide)

### 1. 봇 토큰 발급 (선행 작업)
1. 텔레그램 앱을 켜고 상단 검색창에 **BotFather**를 검색합니다. (공식 파란색 인증 배지 확인)
2. 대화창에 `/newbot` 입력 후, 안내에 따라 봇 이름과 봇 유저네임(반드시 `bot`으로 끝나야 함)을 생성합니다.
3. 생성이 완료되면 발급되는 **HTTP API Token** (예: `1234567890:AAH...`)을 복사하여 보관합니다.

### 2. 코드 다운로드 및 환경 변수 준비
1. 이 프로젝트의 소스 코드를 다운로드(또는 git clone) 받은 뒤, 폴더를 에디터(VSCode 등)로 엽니다.
2. 터미널(터미널 창)에서 프로젝트 루트 폴더 안으로 이동합니다.
   ```bash
   cd 경로/Telegram_Bot
   ```
3. 디렉토리 안에 있는 `.env.example` 파일을 복사하여 **`.env`** 라는 이름의 새 파일을 만듭니다.
4. 새로 만든 `.env` 파일을 열고 다음과 같이 본인의 토큰과 User ID를 기입합니다.
   ```env
   # 발급받은 API 토큰 입력
   TELEGRAM_BOT_TOKEN="여기에_토큰_붙여넣기"

   # 봇을 사용할 수 있는 권한을 가진 텔레그램 User ID
   # (본인의 ID를 모른다면 숫자 아무거나 넣고 하단의 'ID 확인 팁' 참고)
   ADMIN_USER_ID="123456789"
   ```

### 3. 필수 패키지 설치 (가상환경 사용 권장)
파이썬 패키지가 컴퓨터 시스템 꼬임을 유발하지 않도록 가상환경(`venv`) 안에서 실행하는 것을 권장합니다.

1. **가상환경 생성**
   ```bash
   python3 -m venv venv
   ```
2. **가상환경 활성화 (MacOS / Linux)**
   ```bash
   source venv/bin/activate
   ```
   *(윈도우 사용자의 경우: `venv\Scripts\activate`)*
3. **명세서(requirements.txt) 기반 라이브러리 설치**
   ```bash
   pip install -r requirements.txt
   ```

### 4. 봇 실행하기
모든 설치가 끝났습니다! 아래 명령어로 봇을 실행하세요.
```bash
python bot.py
```
성공적으로 켜지면 터미널에 `봇 폴링(Polling)을 시작합니다` 라는 안내 메시지가 노출됩니다.

## 💡 사용 방법 및 ID 확인 팁

- **메시지 전송 방법**
  - 자신이 생성해둔 텔레그램 봇 대화방으로 들어가서 `/get 삼성전자`와 같이 전송해 보세요. 
  - 미국달러, 호주달러, 일본엔화 환율은 `/rate`를 입력하면 확인 할 수 있습니다.

- **내 텔레그램 User ID 알아내기**
  - 만약 아직 본인의 `ADMIN_USER_ID`를 모르신다면, 아무 아이디나 적힌 `.env` 상태에서 봇을 켠 다음 위와 같이 봇에게 말을 걸어보세요. 권한이 막혔다며 **거절 메시지와 동시에 본인의 실제 User ID를 터미널과 채팅방에 알려줍니다.** 그 번호를 복사해서 다시 `.env` 파일에 덮어씌워 주시면 됩니다.
- **아무나 쓰게 만들려면?**
  - `bot.py` 파일 내에 위치한 `if user_id != ADMIN_USER_ID:` 부분을 주석 처리하시면 단톡방이나 여러 명이 쓰는 공개 봇으로 응용하실 수 있습니다.
